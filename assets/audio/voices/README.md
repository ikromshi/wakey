# Voice Sample Audio Files

Place voice sample audio files in this directory.

## Naming Convention

Each voice sample should be named: `{voice-id}-sample.mp3`

## Required Files

- `jen-sample.mp3` - Sample of Jen's voice (British, soothing)
- `jessica-sample.mp3` - Sample of Jessica's voice (calm, garden-soft)
- `milo-sample.mp3` - Sample of Milo's voice (American, relaxed)
- `nathaniel-sample.mp3` - Sample of Nathaniel's voice (British, deep)
